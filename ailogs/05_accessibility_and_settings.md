# 05 - Accessibility & Settings

## Primary Prompt A
Create an accessibility page linked from My > Settings.
Include:
- Font size in three levels: default / medium / large.
- High contrast mode.
- Optional screen-reader assist toggle.
- Preview line to observe style impact.

## Primary Prompt B
Ensure accessibility settings are global and persistent:
- Apply across pages.
- Keep UI stable on mobile after font scaling.

## Primary Prompt C
Restructure My page settings:
- Keep install-to-home guidance.
- Move clear-local-records into an independent section at the bottom.
- Ensure clear-local-records resets global local states consistently.
